# Skull Party

A card game built with React where players flip cards - flowers are good, skulls make you lose cards. Win 2 rounds to win the game!

## Features

- Circular card design
- Face-up cards in collection (you can see your own cards)
- Face-down cards in landing zone
- Round tracking with visual indicators
- Player elimination when out of cards
- Local storage persistence

## Getting Started

### Installation

```bash
npm install
```

### Development

```bash
npm run dev
```

### Build

```bash
npm run build
```
